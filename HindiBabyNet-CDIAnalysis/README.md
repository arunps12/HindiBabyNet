# HindiBabyNet-CDIAnalysis

CDI (MacArthur-Bates Communicative Development Inventories) questionnaire analyses
for the HindiBabyNet project.

## Setup

```bash
uv sync
uv run python -c "import hindibabynet_cdianalysis"
```

## Data

Expects CDI response data under `data/CDI/`.

## Usage

See `notebooks/` for analyses.
